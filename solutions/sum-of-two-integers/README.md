# Sum of Two Integers

Given two integers <code>a</code> and <code>b</code>, return *the sum of the two integers without using the operators* <code>+</code> *and* <code>-</code>.


**Example 1:**
>
> **Input:** a = 1, b = 2
>
> **Output:** 3

**Example 2:**
>
> **Input:** a = 2, b = 3
>
> **Output:** 5


**Constraints:**

- <code>-1000 &lt;= a, b &lt;= 1000</code>
